## Local Development

```sh
# Install packages

npm i
yarn
pnpm i
bun i
```

```sh
# Run code

npm run dev
yarn dev
pnpm run dev
bun run dev
```

### Setup .env file

```sh
# .env or .env.local

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=
```

### Add SQL Tables

Use `database.sql` file

### Start the app

```sh
npm run dev
```

### Run supabase locally

```sh
supabase start
```

```sh
supabase stop
```

```sh
supabase db reset
```
